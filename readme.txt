This plugin allows the admin to select a support ribbon that will be displayed in the lower left corner of your WordPress site.

== Description ==

This plugin adds a support ribbon to your WordPress site.

==For more information visit==

[WordPress Support Ribbon](http://portalplanet.net/wordpress-themes-plugins/support-ribbon-wordpress-plugin/)

== Installation ==

1. Upload the `support-ribbons` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the settings (Admin > Settings > Ribbon Manager)
4. Select the ribbon you would like to be displayed and click save.
5. Go to your site and the ribbon will be displayed at the bottom left corner of your pages/posts.
